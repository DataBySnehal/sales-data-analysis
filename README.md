# Sales Data Analysis Project

## Overview
This project analyzes sales data using Python and Pandas.

## Technologies Used
- Python
- Pandas
- Matplotlib
- Git

## Analysis Performed
- Total Sales Analysis
- Total Profit Analysis
- Sales by Category
- Profit by Category
- Sales by Year
- Top 10 Products Analysis
